北方姑娘在南方姑娘

南方姑娘是蜜桃最近发现的一家宝藏饭店

用饭店来形容它可能不太合适，更像是蜜桃和葡萄的秘密基地

因为店里人很少，来了两次都没碰到其他堂食的顾客

霸占整个二楼的感觉非常爽

老板和老板娘应该是两个很有生活情趣的人

二楼有面墙挂满了拍立得照片，全是他们去过的地方

蜜桃猜测二楼挂着仓库牌子的房间是他们的卧室

葡萄也想和蜜桃过上这样的生活

菜品不是很多，但吃过的都不错

第一次来吃的海苔炒饭有点咸，那天晚上睡前葡萄喝了一大碗水

韩式炸鸡味道不错，但不如无骨炸鸡合适

炸鸡只有两种酱料，蜂蜜芥末和甜辣酱

甜辣酱不如熊家炸鸡的好吃，蜂蜜芥末还不错

蜜桃说丝袜奶茶不好喝，于是今天穿了低胸衣

于是蜜桃今天多了一位球迷

天气之子感觉一般，但和蜜桃一起看电影重要的不是电影

是蜜桃❤

本博客的诞生过程

1.在nodejs.org安装推荐版的nodejs

​ 安装好用node -v和node -v检查一下版本

2.安装淘宝镜像源

​ npm install -g cnpm –registry=https://registry.npm.taobao.org

​ 安装好后输入cnpm检验

3.cnpm install -g hexo-cli

​ hexo -v

4.hexo init

5.hexo s查看预览

6.hexo n “文章名称”

7.cd到source/_posts找到文章，vim 文章名称.md编辑(直接用typora写文章)

8.回到根目录下，hexo clean， hexo g， hexo s

9.部署到github：

9.1新建仓库，命名一定要是github名称.github.io
9.3cnpm install –save hexo-deployer-git
9.4将_config.yml里的
deployment里的tyoe改成git
加入repo: 仓库地址
branch: master
9.5 hexo d部署到github
