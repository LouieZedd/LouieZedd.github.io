Hexo相关配置

安装git

在nodejs.org安装推荐版的nodejs

​ 安装好用node -v和npm -v检查一下版本

安装淘宝镜像源（下载速度更快）

​ npm install -g cnpm –registry=https://registry.npm.taobao.org

​ 安装好后输入cnpm检验

通过cnpm安装hexo

​ cnpm install -g hexo-cli

初始化博客

​ hexo init

查看预览

​ hexo s

新建一篇文章

​ hexo n “文章名称”

找到md格式的文章，用typora编写

文章保存在source/_posts

回到根目录下，hexo clean， hexo g， hexo s

部署到github：

1.新建仓库，命名一定要是github用户名.github.io

2.cnpm install –save hexo-deployer-git

3.将_config.yml里的deployment里的type改成git

加入:

​ repo: 仓库地址，branch: master

4.hexo d部署到github
