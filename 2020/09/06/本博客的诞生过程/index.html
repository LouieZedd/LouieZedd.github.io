1.在nodejs.org安装推荐版的nodejs

​ 安装好用node -v和node -v检查一下版本

2.安装淘宝镜像源

​ npm install -g cnpm –registry=https://registry.npm.taobao.org

​ 安装好后输入cnpm检验

3.cnpm install -g hexo-cli

​ hexo -v

4.hexo init

5.hexo s查看预览

6.hexo n “文章名称”

7.cd到source/_posts找到文章，vim 文章名称.md编辑(直接用typora写文章)

8.回到根目录下，hexo clean， hexo g， hexo s

9.部署到github：

9.1新建仓库，命名一定要是github名称.github.io
9.3cnpm install –save hexo-deployer-git
9.4将_config.yml里的
deployment里的tyoe改成git
加入repo: 仓库地址
branch: master
9.5 hexo d部署到github
